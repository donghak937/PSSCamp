##  문제 풀이

###  문제 링크  
[389. Find the Difference – Leetcode](https://leetcode.com/problems/find-the-difference/description/).

---

###  문제 설명  
> 문자열에서 추가된 한 글자 찾기

---

###  접근 방식  
1. 두 문자열을 sort한다.
2. 다른 것을 return 하거나, 끝까지 없다면 마지막 글자를 리턴한다.
---

### 풀이 코드

```cpp
#include <iostream>
#include <algorithm>
#include <vector>
using namespace std;

int main() {
    string s = "asd";
    string t = "asdr";
    sort(s.begin(), s.end());
    sort(t.begin(), t.end());
    for(int i = 0; i < s.size(); i++){
        if (s[i] != t[i]) return t[i];
    }
    cout << t[t.size() - 1];
}
```
