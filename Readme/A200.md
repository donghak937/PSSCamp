## 문제 풀이

### 문제 링크
[9656. 돌 게임 2 – 백준](https://www.acmicpc.net/problem/9656)

---

### 문제 설명
> 탁자 위에 돌 N개가 있다. 상근이와 창영이는 턴을 번갈아가며 돌을 가져가며, 돌은 1개 또는 3개 가져갈 수 있다. 마지막 돌을 가져가는 사람이 게임을 지게 된다.
> 두 사람이 완벽하게 게임을 했을 때, 이기는 사람을 구하는 프로그램을 작성하시오. 게임은 상근이가 먼저 시작한다.

---

### 접근 방식
1.  **게임 이론 (Game Theory)** 혹은 **동적 계획법 (Dynamic Programming)** 문제지만, 규칙을 찾으면 간단한 수학 문제로 풀 수 있다.
2.  작은 수부터 승패 패턴을 분석해본다. (조건: 마지막 돌을 집으면 **패배**)
    -   `N=1`: 상근이가 1개를 가져가면 패배. -> **창영(CY) 승**
    -   `N=2`: 상근이가 1개를 가져가면 1개 남음. 창영이가 1개를 가져가면 패배. -> **상근(SK) 승**
    -   `N=3`:
        -   상근이가 1개를 가져가면 2개 남음. (N=2 케이스를 창영이가 시작하는 셈 -> 창영 승리). 즉 상근 패배.
        -   상근이가 3개를 가져가면 패배.
        -   결국 **창영(CY) 승**
    -   `N=4`: 상근이가 1개를 가져가면 3개 남음(N=3 케이스 -> 상대방 승리). 즉, 상근이는 3개 남은 상황을 창영이에게 넘기면 이길 수 있음. -> **상근(SK) 승**
3.  **패턴 발견**:
    -   N이 홀수일 때: **창영(CY) 승**
    -   N이 짝수일 때: **상근(SK) 승**
4.  따라서 `N % 2`가 0인지 1인지에 따라 결과를 출력하면 된다.

---

### 풀이 코드

```cpp
#include <iostream>
using namespace std;

int main() {

    long long n;
    cin >> n;

    if (n % 2 == 0) cout << "SK\n";
    else cout << "CY\n";
}
```
