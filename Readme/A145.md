## 문제 풀이

### 문제 링크
[부족한 금액 계산하기 – 프로그래머스](https://school.programmers.co.kr/learn/courses/30/lessons/82612)

---

### 문제 설명
> 놀이기구를 N번 이용하면 원래 이용료의 N배를 받기로 하였습니다. 놀이기구의 처음 이용료가 price일 때, 놀이기구를 count번 타게 되면 현재 자신이 가지고 있는 금액 money에서 얼마가 모자라는지 return 하는 문제. (단, 금액이 부족하지 않으면 0을 return)

---

### 접근 방식
1. 1부터 `count`까지 반복하며 각 횟수별 이용료(`price * i`)를 총액에 더함
2. 전체 이용료 합산 결과가 현재 가진 돈(`money`)보다 크면 그 차액을 반환하고, 아니면 0을 반환
---

### 풀이 코드

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main(){

    int price = 3,money = 20, count = 4;
    long long answer = 0;
    
    for(int i = 1; i <= count; i++){
        answer += price * i;
    }
    if (answer - money < 0) return 0;
    return answer - money;

}
```
