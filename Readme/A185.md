## 문제 풀이

### 문제 링크
[1330. 두 수 비교하기 – 백준](https://www.acmicpc.net/problem/1330)

---

### 문제 설명
> 두 정수 A와 B가 주어졌을 때, A와 B를 비교하는 프로그램을 작성하시오.

---

### 접근 방식
1.  **조건문 사용**:
    -   두 정수 `a`, `b`를 입력받는다.
    -   `if` 문을 사용하여 크기를 비교한다.
    -   `a > b`이면 `>` 출력
    -   `a < b`이면 `<` 출력
    -   둘 다 아니라면(같다면) `==` 출력
2.  **주의**:
    -   각 출력 뒤에 줄바꿈(`endl` 또는 `\n`)을 포함한다. (문제 조건에 따라 필수일 수도 있고 아닐 수도 있으나, 일반적으로는 해주는 것이 좋다.)

---

### 풀이 코드

```cpp
#include <iostream>

using namespace std;

int main(){
    int a, b;

    cin >> a >> b;

    if (a > b) cout << ">" << endl;
    else if (a < b) cout << "<" << endl;
    else cout << "==" << endl;

    return 0;
}
```
