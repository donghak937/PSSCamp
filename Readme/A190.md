## 문제 풀이

### 문제 링크
[2480. 주사위 세개 – 백준](https://www.acmicpc.net/problem/2480)

---

### 문제 설명
> 1에서부터 6까지의 눈을 가진 3개의 주사위를 던져서 다음과 같은 규칙에 따라 상금을 받는 게임이 있다.
> 1.  같은 눈이 3개가 나오면 10,000원+(같은 눈)×1,000원의 상금을 받게 된다.
> 2.  같은 눈이 2개만 나오면 1,000원+(같은 눈)×100원의 상금을 받게 된다.
> 3.  모두 다른 눈이 나오면 (그 중 가장 큰 눈)×100원의 상금을 받게 된다.
>
> 3개의 주사위 눈이 주어질 때, 상금을 계산하는 프로그램을 작성하시오.

---

### 접근 방식
1.  **조건문 사용**:
    -   입력반은 세 주사위의 눈 `d1, d2, d3`를 비교한다.
    -   가장 큰 눈(`max`)을 찾는다. (세 눈이 다를 때 사용)
2.  **경우의 수 나누기**:
    -   **세 개 모두 같은 경우**: `d1 == d2 && d2 == d3`
        -   `10000 + d1 * 1000`
    -   **두 개만 같은 경우**: 위 조건이 아닐 때 `d1 == d2 || d1 == d3 || d2 == d3`
        -   같은 눈을 찾아 `1000 + (같은 눈) * 100` 계산
    -   **모두 다른 경우**:
        -   `max * 100` 계산

---
