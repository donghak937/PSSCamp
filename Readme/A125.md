##  문제 풀이

###  문제 링크  
[2217. 로프 – 백준](https://www.acmicpc.net/problem/2217)

---

###  문제 설명  
> 줄을 연결해서 만들 수 있는 가장 무거운 무게 찾기

---

###  접근 방식  
1. 사용하지 않아도 되는 줄이 있으니, 그것을 이용한다.
2. 가장 큰 수 부터 n-1씩 해가며 가장 무거운 구간을 찾는다.
---

### 풀이 코드

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main() {
    int n;
    vector<int> ropes;
    cin >> n;
    for(int i = 0; i < n; i ++){
        int a;
        cin >> a;
        ropes.push_back(a);
    }
    sort(ropes.begin(), ropes.end());
    int max = 0;

    for(int i = ropes.size()-1; i >= 0; i--){
        if (ropes[i] * (n - i) > max){
            max = ropes[i] * (n - i);
        }
    }
    cout << max;

}
```
