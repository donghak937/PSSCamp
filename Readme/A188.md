## 문제 풀이

### 문제 링크
[14681. 사분면 고르기 – 백준](https://www.acmicpc.net/problem/14681)

---

### 문제 설명
> 흔한 수학 문제 중 하나는 주어진 점이 어느 사분면에 속하는지 알아내는 것이다. 사분면은 아래 그림처럼 1부터 4까지 번호를 갖는다. "Quadrant n"은 "제n사분면"이라는 뜻이다.
> 예를 들어, 좌표가 (12, 5)인 점 A는 x좌표와 y좌표가 모두 양수이므로 제1사분면에 속한다. 점 B는 x좌표가 음수이고 y좌표가 양수이므로 제2사분면에 속한다.
> 점의 좌표를 입력받아 그 점이 어느 사분면에 속하는지 알아내는 프로그램을 작성하시오. 단, x좌표와 y좌표는 모두 0이 아니다.

| 사분면 | x 좌표 부호 | y 좌표 부호 |
| :--: | :-------: | :-------: |
| 1 | + (양수) | + (양수) |
| 2 | - (음수) | + (양수) |
| 3 | - (음수) | - (음수) |
| 4 | + (양수) | - (음수) |

---

### 접근 방식
1.  **조건문 사용**:
    -   `x`와 `y`를 입력받는다.
    -   `x > 0` 이면서 `y > 0` 이면 **1사분면**
    -   `x < 0` 이면서 `y > 0` 이면 **2사분면**
    -   `x < 0` 이면서 `y < 0` 이면 **3사분면**
    -   나머지 경우 (`x > 0`, `y < 0`) 이면 **4사분면**
2.  **구현 로직**:
    -   `if (x > 0 && y > 0)` -> 1
    -   `else if (x <= 0 && y > 0)` -> 2 (문제 조건상 좌표는 0이 아니지만, 코드에서는 포괄적으로 처리함)
    -   `else if (x <= 0 && y <= 0)` -> 3
    -   `else` -> 4

---

### 풀이 코드

```cpp
#include <iostream>
using namespace std;

int main(){
    int x, y;
    
    cin >> x >> y;
    
    if (x > 0 && y > 0) {
        cout << "1";
    } else if (x <= 0 && y > 0) {
        cout << "2";
    } else if (x <= 0 && y <= 0) {
        cout << "3";
    } else {
        cout << "4";
    }
    
    return 0;
}
```
