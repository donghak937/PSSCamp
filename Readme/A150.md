## 문제 풀이

### 문제 링크
[1912. 연속합 – 백준](https://www.acmicpc.net/problem/1912)

---

### 문제 설명
> n개의 정수로 이루어진 임의의 수열이 주어진다. 우리는 이 중 연속된 몇 개의 수를 선택해서 구할 수 있는 합 중 가장 큰 합을 구하려고 한다. 단, 수는 한 개 이상 선택해야 한다.

---

### 접근 방식
1.  **동적 계획법 (Dynamic Programming) - 카데인 알고리즘 (Kadane's Algorithm)** 사용
2.  `now` 변수에 "현재 위치까지의 연속 합 중 최댓값"을 저장한다.
    -   이전까지의 합(`now + a`)과 현재 값(`a`) 중 더 큰 값을 선택하여 갱신한다.
    -   즉, `now = max(a, now + a)`
3.  `maxi` 변수에 "지금까지 발견된 연속 합 중 최댓값"을 저장한다.
    -   매 반복마다 `maxi`를 `now`와 비교하여 갱신한다.
4.  최종적으로 `maxi`를 출력한다.

---

### 풀이 코드

```cpp
#include <iostream>
#include <vector>

using namespace std;

int main() {

    int n;
    cin >> n;

    int now;
    int maxi;

    for(int i = 0; i < n; i++){
        int a;
        cin >> a;

        if (!i){
            now = a;
            maxi = a;
        }

        else{
            now = max(a, now + a);
            maxi = max(now, maxi);
        }


    }

    cout << maxi;
}
```
