## 문제 풀이

### 문제 링크
[10430. 나머지 – 백준](https://www.acmicpc.net/problem/10430)

---

### 문제 설명
> (A+B)%C는 ((A%C) + (B%C))%C 와 같을까?
> (A×B)%C는 ((A%C) × (B%C))%C 와 같을까?
> 세 수 A, B, C가 주어졌을 때, 위의 네 가지 값을 구하는 프로그램을 작성하시오.

---

### 접근 방식
1.  **연산**:
    -   문제에서 요구하는 네 가지 수식을 그대로 코드로 구현한다.
    -   1. `(a + b) % c`
    -   2. `((a % c) + (b % c)) % c`
    -   3. `(a * b) % c`
    -   4. `((a % c) * (b % c)) % c`
2.  **출력**:
    -   각 결과를 한 줄에 하나씩 출력한다.

---

### 풀이 코드

```cpp
#include <iostream>
using namespace std;

int main(){
    int a, b, c;
    cin >> a >> b >> c;
    cout << (a+b)%c << "\n";
    cout << ((a%c) + (b%c))%c << "\n";
    cout << (a*b)%c << "\n";
    cout << ((a%c) * (b%c))%c << "\n";
}
```
