## 문제 풀이

### 문제 링크
[1300. K번째 수 – 백준](https://www.acmicpc.net/problem/1300)

---

### 문제 설명
> N×N 크기의 배열 A가 있고, A[i][j] = i×j이다. 이 수를 일차원 배열 B에 넣고 오름차순 정렬했을 때, B[k]를 구하는 문제

---

### 접근 방식
1. K번째 수 x를 이분 탐색으로 찾음
2. 각 행 i에서 x보다 작거나 같은 수의 개수는 min(N, x/i)
3. 모든 행의 개수 합이 K보다 크거나 같은 최소 x를 구함
---

### 풀이 코드

```cpp
#include <iostream>
#include <algorithm>

using namespace std;

int main() {
    long long N, K;
    cin >> N >> K;

    long long lo = 1, hi = K; 
    while (lo < hi) {
        long long mid = (lo + hi) / 2;

        long long cnt = 0;
        for (long long i = 1; i <= N; i++) {
            cnt += min(N, mid / i);
        }

        if (cnt >= K) hi = mid;      
        else lo = mid + 1;          
    }

    cout << lo << "\n";
}
```
