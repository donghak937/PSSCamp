##  문제 풀이

###  문제 링크  
[9095. 1, 2, 3 더하기 – 백준](https://www.acmicpc.net/problem/9095)

---

###  문제 설명  
> 숫자를 1, 2, 3을 더해서 만든다. 

---

###  접근 방식  
1. 다이나믹 프로그래밍을 사용해서 만든다.
---

### 풀이 코드

```cpp
#include <iostream>

using namespace std;

int main() {
    int N;
    int dp[12];
    cin >> N;
    dp[1] = 1;
    dp[2] = 2;
    dp[3] = 4;
    for(int i = 4; i <= 12; i++){
        dp[i] = dp[i-1] + dp[i-2] + dp[i-3];
    }

    for(int i = 0; i < N; i++){
        int a;
        cin >> a;
        cout << dp[a] << '\n';
    }

}
```
