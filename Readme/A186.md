## 문제 풀이

### 문제 링크
[9498. 시험 성적 – 백준](https://www.acmicpc.net/problem/9498)

---

### 문제 설명
> 시험 점수를 입력받아 90 ~ 100점은 A, 80 ~ 89점은 B, 70 ~ 79점은 C, 60 ~ 69점은 D, 나머지 점수는 F를 출력하는 프로그램을 작성하시오.

---

### 접근 방식
1.  **조건문 비교**:
    -   입력된 점수(`score`)가 90 이상인지 확인한다.
    -   아니라면 80 이상인지 확인한다. (이미 90 미만임이 보장됨)
    -   아니라면 70 이상인지 확인한다.
    -   아니라면 60 이상인지 확인한다.
    -   그 외의 경우는 모두 F를 출력한다.
2.  **주의**:
    -   점수는 0보다 크거나 같고, 100보다 작거나 같은 정수이다.

---

### 풀이 코드

```cpp
#include <iostream>

using namespace std;

int main(){
    
  int score;
  
  cin >> score;

  if (score >= 90){
    cout << "A";
  }
  else if (score >= 80){
    cout << "B";
  }
  else if (score >= 70){
    cout << "C";
  }
  else if (score >= 60){
    cout << "D";
  }
  else{
    cout << "F";
  }
  
  return 0;
}
```
