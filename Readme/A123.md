##  문제 풀이

###  문제 링크  
[1463. 1로 만들기 – 백준](https://www.acmicpc.net/problem/1463)

---

###  문제 설명  
> 문자 n을 2, 3, 1을 사용해서 1로 만들기

---

###  접근 방식  
1. DP를 사용했다.
2. 모든 접근 방법을 사용한다.

---

### 풀이 코드

```cpp
#include <iostream>

using namespace std;

int main() {
    int dp[1000001] = {0};

    dp[1] = 0;

    int n;
    cin >> n;

    for (int i = 2; i <= n; i++) {
        dp[i] = dp[i-1] + 1;

        if (i % 2 == 0)
            dp[i] = min(dp[i], dp[i/2] + 1);

        if (i % 3 == 0)
            dp[i] = min(dp[i], dp[i/3] + 1);
    }

    cout << dp[n];
}
```
