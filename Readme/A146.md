## 문제 풀이

### 문제 링크
[최솟값 만들기 – 프로그래머스](https://school.programmers.co.kr/learn/courses/30/lessons/12941)

---

### 문제 설명
> 길이가 같은 배열 A, B 두 개가 있습니다. 각 배열에서 숫자를 하나씩 뽑아 두 숫자를 곱합니다. 이러한 과정을 배열의 길이만큼 반복하며, 두 숫자를 곱한 값을 누적하여 더합니다. 최종적으로 누적된 값이 최소가 되도록 만드는 것이 목표입니다.

---

### 접근 방식
1. 누적 합을 최소로 만들기 위해서는 한 배열의 가장 작은 값과 다른 배열의 가장 큰 값을 곱해야 함
2. 배열 A는 오름차순으로 정렬 (`sort`)
3. 배열 B는 내림차순으로 정렬 (`sort`와 `greater<int>()` 사용)
4. 각 인덱스별로 두 값을 곱하여 합산
---

### 풀이 코드

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main(){

    vector<int> A = {1, 4, 2};
    vector<int> B = {5, 4, 4};

    int answer = 0;

    sort(A.begin(), A.end());
    sort(B.begin(), B.end(), greater<int>());
    
    for(int i = 0; i < A.size(); i++){
        answer += A[i] * B[i];
    }
    

    return answer;

}
```
