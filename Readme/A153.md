## 문제 풀이

### 문제 링크
[11399. ATM – 백준](https://www.acmicpc.net/problem/11399)

---

### 문제 설명
> 줄을 서 있는 사람들의 수 N과 각 사람이 돈을 인출하는 데 걸리는 시간 P가 주어졌을 때, 각 사람이 돈을 인출하는 데 필요한 시간의 합의 최솟값을 구하는 문제. 앞사람의 시간이 뒷사람에게 누적되므로, 시간이 적게 걸리는 사람부터 처리하는 것이 총 대기 시간을 줄이는 방법이다.

---

### 접근 방식
1. 입력받은 인출 시간들을 오름차순으로 정렬한다.
2. 각 사람의 대기 시간(이전 사람들의 시간 합 + 본인 시간)을 계산하여 누적한다.
3. 누적된 총합을 출력한다.

---

### 풀이 코드

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main() {
    int n;
    vector<int> a;

    cin >> n;

    int time = 0;
    int total = 0;

    for(int i = 0; i < n; i++){
        int b; 
        cin >> b;
        a.push_back(b);
    }
    sort(a.begin(), a.end());

    for(int i = 0; i < n; i++){
        time += a[i];
        total += time;
    }

    cout << total;

}
```
