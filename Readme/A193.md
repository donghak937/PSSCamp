## 문제 풀이

### 문제 링크
[25314. 코딩은 체육과목 입니다 – 백준](https://www.acmicpc.net/problem/25314)

---

### 문제 설명
> 오늘은 혜아의 면접 날이다. 면접관실에 들어간 혜아는 칠판에 적힌 문제를 풀라는 지시를 받았다.
> 문제: `long int`는 4바이트 정수까지 저장할 수 있는 정수 자료형이고, `long long int`는 8바이트 정수까지 저장할 수 있는 정수 자료형이다.
> `long long long int`는 12바이트 정수까지 저장할 수 있는 정수 자료형이고, `long long long long int`는 16바이트 정수까지 저장할 수 있는 정수 자료형이다.
> 이와 같이, `N`바이트 정수까지 저장할 수 있는 정수 자료형의 이름에 포함된 `long`의 개수를 구하는 프로그램을 작성하시오.
> (`long`은 `N`을 4로 나눈 몫만큼 반복되어 출력되며, 마지막에는 `int`가 붙는다.)

---

### 접근 방식
1.  **반복문 사용**:
    -   입력받은 `n`은 4의 배수이다.
    -   `n`을 4로 나눈 값(`n / 4`)만큼 `long`이라는 문자열을 반복해서 출력한다.
    -   마지막에 `int`를 출력한다.
    -   예) `n=20`이면, `20 / 4 = 5`번 `long`을 출력하고 `int`를 붙여 `long long long long long int`가 된다.
2.  **구현 로직**:
    -   `n` 입력 받기
    -   `n /= 4`
    -   `n`번 만큼 for문을 돌면서 `cout << "long ";`
    -   loop 종료 후 `cout << "int";`

---

### 풀이 코드

```cpp
#include <iostream>

using namespace std;

int main() {
    int n;
    cin >> n;
    n /= 4;
    
    for (int i = 0; i < n; i++) {
        cout << "long ";
    }
    cout << "int";
    
    return 0;
}
```
