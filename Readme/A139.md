## 문제 풀이

### 문제 링크
[1439. 뒤집기 – 백준](https://www.acmicpc.net/problem/1439)

---

### 문제 설명
> 0과 1로만 이루어진 문자열 S가 주어졌을 때, 모든 숫자를 같게 만들기 위해 뒤집어야 하는 최소 횟수를 구하는 문제. (한 번 뒤집을 때 연속된 하나 이상의 숫자를 뒤집을 수 있음)

---

### 접근 방식
1. 연속된 0의 묶음 개수와 1의 묶음 개수를 각각 카운트
2. 이전 문자와 다른 문자가 나타날 때마다 해당 문자의 묶음 개수를 증가
3. 두 묶음 개수 중 더 적은 쪽이 최소 뒤집기 횟수가 됨
---

### 풀이 코드

```cpp
#include <iostream>
#include <string>
#include <algorithm>

using namespace std;

int main() {
    string s;
    cin >> s;

    int oneCnt = 0;
    int zeroCnt = 0;

    if (s[0] == '0') zeroCnt++;
    else oneCnt++;

    for(int i = 0; i < s.size()-1; i++){
        
        if(s[i] == s[i+1]) continue;
        else {
            if(s[i+1] == '1') oneCnt++;
            else zeroCnt++;
        }

    }

    cout << min(zeroCnt, oneCnt);

}
```
