## 문제 풀이

### 문제 링크
[11382. 꼬마 정민 – 백준](https://www.acmicpc.net/problem/11382)

---

### 문제 설명
> 꼬마 정민이는 이제 A + B 정도는 쉽게 계산할 수 있다. 이제 A + B + C를 계산할 차례이다!
> 첫 번째 알, A, B, C (1 ≤ A, B, C ≤ $10^{12}$)를 입력받는다.
> A+B+C의 값을 출력한다.

---

### 접근 방식
1.  **자료형 선택**:
    -   입력되는 A, B, C의 최대 크기는 $10^{12}$이다.
    -   일반적인 `int` 자료형은 약 $2 \times 10^9$까지만 저장할 수 있으므로, 범위를 초과한다.
    -   따라서 64비트 정수형인 `long long`을 사용해야 한다.
2.  **입출력**:
    -   세 수를 입력받아 단순 합을 출력한다.

---

### 풀이 코드

```cpp
#include <iostream>
using namespace std;

int main(){
    long long int a, b, c;
    cin >> a >> b >> c;
    cout << a + b + c;
}
```
