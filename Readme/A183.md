## 문제 풀이

### 문제 링크
[10171. 고양이 – 백준](https://www.acmicpc.net/problem/10171)

---

### 문제 설명
> 아래 예제와 같이 고양이를 출력하시오.

**예제 출력**
```
\    /\
 )  ( ')
(  /  )
 \(__)|
```

---

### 접근 방식
1.  **이스케이프 시퀀스(Escape Sequence)** 처리:
    -   역슬래시(`\`)와 같은 특수 문자를 출력하기 위해 앞에 역슬래시를 하나 더 붙여야 한다.
    -   `\` 출력 -> `\\`
    -   작은따옴표(`'`)는 그대로 출력해도 되지만, 안전하게 `\'`로 쓰는 경우도 있다. (C++에서는 `cout` 내부에선 그냥 `'` 써도 됨)
    -   괄호 `(`, `)` 등은 그대로 출력 가능하다.
2.  **줄바꿈**:
    -   각 줄 끝에 `\n`을 넣어 줄바꿈을 한다.

---

### 풀이 코드

```cpp
#include <iostream>
using namespace std;

int main(){
    cout << "\\    /\\\n";
    cout << " )  ( \')\n";
    cout << "(  /  )\n";
    cout << " \\(__)|";
}
```
