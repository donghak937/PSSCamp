##  문제 풀이

###  문제 링크  
[10773. 제로 – 백준](https://www.acmicpc.net/problem/10773)

---

###  문제 설명  
> 숫자를 쭉 입력하다가 0나오면 빼기

---

###  접근 방식  
1. 스택에 넣는다.
2. 0나오면 뺀다.
---

### 풀이 코드

```cpp
#include <iostream>
#include <stack>

using namespace std;

int main() {

    stack<int> a;
    int n;
    cin >> n;

    for(int i = 0; i < n; i++){
        int tmp;
        cin >> tmp;
        if (tmp == 0) a.pop();
        else a.push(tmp);
    }

    int sum = 0;
    while (!a.empty()) {
        sum += a.top();
        a.pop();
    }
    cout << sum << endl;
    
}
```
