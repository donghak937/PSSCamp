## 문제 풀이

### 문제 링크
[10807. 개수 세기 – 백준](https://www.acmicpc.net/problem/10807)

---

### 문제 설명
> 총 N개의 정수가 주어졌을 때, 정수 v가 몇 개 인지 구하는 프로그램을 작성하시오.

---

### 접근 방식
1.  **배열 순회**:
    -   입력받은 `N`만큼 반복하여 정수들을 배열(`a`)에 저장한다.
    -   찾으려는 정수 `find_num`을 입력받는다.
    -   다시 배열을 순회하면서 `a[i]`가 `find_num`과 같은지 확인한다.
    -   같다면 카운트(`count`)를 1 증가시킨다.
2.  **출력**:
    -   최종적으로 계산된 `count`를 출력한다.

---

### 풀이 코드

```cpp
#include <iostream>

using namespace std;

int main() {
    int size;
    int find_num;
    int a[100] = {0};
    int count = 0;
    
    cin >> size;
    for (int i = 0; i < size; i++) {
        cin >> a[i];
    }
    cin >> find_num;

    for (int i = 0; i < size; i++) {
        if (find_num == a[i]) {
            count++;
        }
    }
    cout << count;
    
    return 0;
}
```
