## 문제 풀이

### 문제 링크
[2588. 곱셈 – 백준](https://www.acmicpc.net/problem/2588)

---

### 문제 설명
> (세 자리 수) × (세 자리 수)는 다음과 같은 과정을 통하여 이루어진다.
> ![곱셈 과정 이미지](https://www.acmicpc.net/upload/images/f5DjyKQY.png)
> (1)과 (2)위치에 들어갈 세 자리 자연수가 주어질 때 (3), (4), (5), (6)위치에 들어갈 값을 구하는 프로그램을 작성하시오.

---

### 접근 방식
1.  **입력 처리**:
    -   첫 번째 수(`a`)와 두 번째 수(`b`)는 정수로 입력받는다.
2.  **각 자리수 분리**:
    -   곱셈 과정을 시뮬레이션하기 위해 두 번째 수(`b`)의 1의 자리, 10의 자리, 100의 자리 숫자가 필요하다.
    -   `b % 10` -> 1의 자리
    -   `(b % 100) / 10` -> 10의 자리
    -   `b / 100` -> 100의 자리
3.  **출력**:
    -   (3): `a * (1의 자리)`
    -   (4): `a * (10의 자리)`
    -   (5): `a * (100의 자리)`
    -   (6): `a * b` (전체 곱셈 결과)
4.  **사용자 코드 방식**:
    -   `b`에서 나머지 연산(`% 10`)과 나눗셈 연산(`-`, `/`)을 반복하여 각 자리수를 구하고 있다.
    -   처음 `b % 10`을 통해 1의 자리를 구하고 `a`와 곱해 출력한다. 그 후 `b`에서 해당 자리를 제거한다.
    -   그 다음 `b % 100`을 통해 10의 자리(의 10배수 형태)를 구하고 처리한다.

---

### 풀이 코드

```cpp
#include <iostream>
using namespace std;

int main(){
    int a, b, n, t;
    cin >> a >> b;
    t = a * b; // 최종 결과 저장
    
    // (3) 출력: 1의 자리 숫자와의 곱
    n = 10;
    n = b % n; 
    b = b - n; 
    cout << a * n << "\n";
    
    // (4) 출력: 10의 자리 숫자와의 곱
    n = 100;
    n = b % n; 
    b = b - n; 
    cout << (a * n) / 10 << "\n"; // n은 10의 자리 수 * 10 형태이므로 10으로 나눔
    
    // (5) 출력: 100의 자리 숫자와의 곱
    n = 100;
    n = b / n;
    cout << a * n << "\n";
    
    // (6) 출력: 전체 곱셈 결과
    cout << t;
}
```
