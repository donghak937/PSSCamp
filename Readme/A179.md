## 문제 풀이

### 문제 링크
[18108. 1998년생인 내가 태국에서는 2541년생?! – 백준](https://www.acmicpc.net/problem/18108)

---

### 문제 설명
> ICPC Bangkok Regional에 참가하기 위해 태국에 갔던 상근이는 한국에 귀국했다.
> 태국은 서기 연도가 아닌 불기 연도를 사용한다. 불기 연도가 주어질 때 이를 서기 연도로 바꿔 주는 프로그램을 작성하시오.

---

### 접근 방식
1.  **연도 변환**:
    -   태국에서 사용하는 불기 연도는 서기 연도보다 **543년** 앞선다.
    -   따라서 입력받은 불기 연도(`y`)에서 **543**을 빼주면 서기 연도를 구할 수 있다.
2.  **입출력**:
    -   불기 연도 `y`를 입력받는다. ($1000 \le y \le 3000$)
    -   `y - 543`을 결과로 출력한다.

---

### 풀이 코드

```cpp
#include <iostream>
using namespace std;

int main(){
    int a;
    cin >> a;
    cout << a - 543;
    
    return 0;
}
```
