## 문제 풀이

### 문제 링크
[10996. 별 찍기 - 21 – 백준](https://www.acmicpc.net/problem/10996)

---

### 문제 설명
> 예제를 보고 규칙을 유추한 뒤에 별을 찍으세요.
> 첫째 줄에 N(1 ≤ N ≤ 100)이 주어졌을 때, 규칙에 따라 별을 출력하는 문제입니다.

**예제 패턴 (N=3인 경우)**
```
* *
 *
* *
 *
* *
 *
```

---

### 접근 방식
1.  **패턴 분석**:
    -   입력 `N`에 대해 총 `2 * N` 줄이 출력된다.
    -   각 줄은 홀수 번째와 짝수 번째 줄이 반복되는 형태다.
    -   **홀수 번째 줄**: 별(`*`)이 먼저 나오고, 한 칸씩 띄엄띄엄 출력된다. (인덱스 0, 2, 4...)
    -   **짝수 번째 줄**: 공백(` `)이 먼저 나오고, 그 다음 별이 출력된다. (인덱스 1, 3, 5...)
2.  **구현 로직**:
    -   총 `N`번의 큰 반복을 수행한다. 한 번의 반복에서 **윗줄(홀수 줄 패턴)과 아랫줄(짝수 줄 패턴)** 을 한 세트로 출력한다.
    -   윗줄: `N`개의 칸 중 짝수 인덱스(`0, 2, 4...`)에 별을 찍고, 나머지는 공백을 찍지 않고 바로 다음 별로 넘어간다. (단, 문제의 예제를 보면 공백 처리가 조금 다를 수 있으나, 일반적으로는 별과 공백을 번갈아 찍는 형태다.)
    -   이 코드는 문제 의도와 달리 직사각형 형태로 찍히는 로직(`for` 루프 구조상 `N x N`)으로 보일 수 있으나, 줄바꿈 처리에 따라 모양이 결정된다.
    -   **사용자 코드 분석**:
        -   `N`번 반복하는 바깥 루프 `i`
        -   안쪽 루프 전에 `i % 2 == 1`일 때 공백을 먼저 출력한다.
        -   안쪽 루프 `j`에서 `N`번 별을 찍고, 사이사이에 공백을 넣는다.
    -   *참고*: 현재 사용자의 코드는 백준 10996번의 정석적인 출력(지그재그 형태)과는 조금 다르게, `N x N` 격자에 별을 찍는 형태에 가깝게 구현되어 있다. (실제 10996번은 `N`이 1일 때 예외 처리 등이 필요하고 줄바꿈 로직이 다름). 하지만 요청에 따라 **사용자의 코드를 있는 그대로 문서화**한다.

---

### 풀이 코드

```cpp
#include <iostream>
using namespace std;

int main() {
    int N;
    cin >> N;
    for (int i = 0; i < N; i++) {
        if (i % 2 == 1) cout << ' ';    
        for (int j = 0; j < N; j++) {
            cout << "*";
            if (j != N - 1) cout << ' ';
            else cout << '\n';
        }
    }
}
```
