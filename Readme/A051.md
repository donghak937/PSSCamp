##  문제 풀이

###  문제 링크  
[5622. 다이얼 – 백준](https://www.acmicpc.net/problem/5622)

---

###  문제 설명  
> 전화 수화기 다이얼에 대응되는 숫자를 더해서 전화번호를 구하는 문제.
---

###  접근 방식  
1. sectocnt 함수를 만든다
2. char과 대응되는 숫자를 리턴하고 더한다.
---

### 풀이 코드

```cpp
#include <iostream>
#include <string>
using namespace std;

int sectocnt(char a) {
    if (a >= 'A' && a <= 'C') return 3;
    if (a >= 'D' && a <= 'F') return 4;
    if (a >= 'G' && a <= 'I') return 5;
    if (a >= 'J' && a <= 'L') return 6;
    if (a >= 'M' && a <= 'O') return 7;
    if (a >= 'P' && a <= 'S') return 8;
    if (a >= 'T' && a <= 'V') return 9;
    if (a >= 'W' && a <= 'Z') return 10;
    return 0; 
}

int main() {
    string a;
    int total = 0;
    cin >> a;

    for (int i = 0; i < a.size(); i++) {
        total += sectocnt(a[i]);
    }

    cout << total;
}




```
