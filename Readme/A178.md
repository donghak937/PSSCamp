## 문제 풀이

### 문제 링크
[1620. 나는야 포켓몬 마스터 이다솜 – 백준](https://www.acmicpc.net/problem/1620)

---

### 문제 설명
> 포켓몬 도감을 완성하기 위해, 포켓몬의 이름("Pikachu")이 입력되면 번호를 출력하고, 번호(25)가 입력되면 포켓몬의 이름을 출력하는 프로그램을 작성하시오.
> 첫째 줄에는 도감에 수록되어 있는 포켓몬의 개수 N(1 ≤ N ≤ 100,000)과 내가 맞춰야 하는 문제의 개수 M(1 ≤ M ≤ 100,000)이 주어집니다.

---

### 접근 방식
1.  **자료구조 선택**:
    -   이름 → 번호: `map<string, int> pocket` (문자열을 키로, 정수를 값으로)
    -   번호 → 이름: `map<int, string> nums` (정수를 키로, 문자열을 값으로)
    -   입력 범위가 100,000개 이므로, `O(1)` 또는 `O(log N)` 시간에 검색이 가능해야 합니다. `map`은 `O(log N)`을 보장합니다.
2.  **입력 최적화**:
    -   C++의 `cin, cout`은 속도가 느릴 수 있으므로 `ios::sync_with_stdio(false); cin.tie(0);`을 사용하여 입출력 속도를 높입니다.
    -   `endl` 대신 `\n`을 사용합니다.
3.  **퀴즈 해결**:
    -   입력된 퀴즈가 숫자인지 문자인지 판별해야 합니다.
    -   `isdigit()` 함수를 사용하여 첫 글자가 숫자인지 확인합니다.
    -   숫자라면 `stoi()`로 변환하여 `nums` 맵에서 이름을 찾아 출력합니다.
    -   문자라면 `pocket` 맵에서 번호를 찾아 출력합니다.

---

### 풀이 코드

```cpp
#include <iostream>
#include <map>
#include <cctype>
#include <string>

using namespace std;

int main(){
    ios::sync_with_stdio(false);
    cin.tie(0);
    cout.tie(0);

    int n, m;

    cin >> n >> m;

    map<string, int> pocket;
    map<int, string> nums;
    for(int i = 1; i <= n; i++){
        string a;
        cin >> a;
        pocket.insert({a, i});
        nums.insert({i, a});
    }

    for(int i = 0; i < m; i++){
        string b;
        cin >> b;

        if (isdigit(b[0])) {
            int c = stoi(b);
            cout << nums[c] << '\n';
        }
        else cout << pocket[b] << '\n';
    }
}
```
