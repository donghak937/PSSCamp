## 문제 풀이

### 문제 링크
[8393. 합 – 백준](https://www.acmicpc.net/problem/8393)

---

### 문제 설명
> n이 주어졌을 때, 1부터 n까지 합을 구하는 프로그램을 작성하시오.

---

### 접근 방식
1.  **반복문 사용**:
    -   입력 `n`까지 반복하는 루프를 만든다.
    -   `total` 변수에 현재 반복자 `i`를 누적하여 더한다.
    -   루프가 끝나면 `total`을 출력한다.
2.  **초기화**:
    -   합을 저장할 변수 `total`은 0으로 초기화해야 한다.
3.  **범위**:
    -   $1 \le n \le 10,000$ 이므로 합은 `int` 범위 내에 충분히 들어온다. (최대 약 5천만)

---

### 풀이 코드

```cpp
#include <iostream>
using namespace std;

int main() {
    int num;
    int total = 0;
    cin >> num;

    for (int i = 0; i <= num; i++) {
        total += i;
    }
    cout << total;
}
```
