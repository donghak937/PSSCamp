## 문제 풀이

### 문제 링크
[1966. 프린터 큐 – 백준](https://www.acmicpc.net/problem/1966)

---

### 문제 설명
> 문서들이 중요도에 따라 인쇄되는 프린터 큐에서, 특정 문서가 몇 번째로 인쇄되는지 구하는 문제. 중요도가 더 높은 문서가 있다면 현재 문서를 가장 뒤로 보냄.

---

### 접근 방식
1. 큐(queue)에는 문서의 인덱스와 중요도를 저장
2. 우선순위 큐(priority_queue)에는 중요도를 저장하여 현재 가장 높은 중요도를 파악
3. 큐의 맨 앞 문서의 중요도가 가장 높으면 인쇄(pop), 아니면 뒤로 보냄(push)
4. 내가 찾는 인덱스의 문서가 인쇄될 때의 순서를 출력
---

### 풀이 코드

```cpp
#include <iostream>
#include <queue>
#include <algorithm>

using namespace std;

int main() {
    int size;
    cin >> size;
    for(int i = 0; i < size; i ++){
        int n, m;
        cin >> n >> m;
        int saved = 0;
        queue<pair<int, int>> q;
        priority_queue<int> pq;
        for(int j = 0; j < n; j++){
            int a;
            cin >> a;
            q.push({j, a});
            pq.push(a);
        }

        while (!pq.empty()) {
            int t = q.front().first;
            int y = q.front().second;
            q.pop();
            if (pq.top() == y){
                saved++;
                pq.pop();
                if(t == m){
                    cout << saved << '\n';
                    break;
                }
            }
            else q.push({t, y});
        }
        
    }
}
```
