##  문제 풀이

###  문제 링크  
[1934. 최소공배수 – 백준](https://www.acmicpc.net/problem/1934)

---

###  문제 설명  
> 최소공배수

---

###  접근 방식  
1. n, m 둘 중에 큰 수를 찾는다.
2. 곱한 다음에 최소까지 나눈다.
---

### 풀이 코드

```cpp
#include <iostream>

using namespace std;

int main() {
	int c;
    cin >> c;

    for(int i = 0; i < c; i ++){
        int n, m;
        cin >> n >> m;

        int a = 1;

        if (n > m){
            for(int i = n*m; i >= n; i--){
                if (i % m == 0 && i % n == 0) a = i;
            }
        }
        else{
            for(int i = n*m; i >= m; i--){
                if (i % m == 0 && i % n == 0) a = i;
            }
        }
        cout << a << '\n';
    }
}
```
