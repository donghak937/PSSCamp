##  문제 풀이

###  문제 링크  
[1934. 최소공배수 – 백준](https://www.acmicpc.net/problem/1934)

---

###  문제 설명  
> 최소공배수

---

###  접근 방식  
1. 쭉 나누어 가면서 모든 한 수를 찾는다.
---

### 풀이 코드

```cpp
#include <iostream>

int count(int n) {
	int hansu = 0;
	int a, b, c;
	for (int i = 1; i <= n; i++) {
		if (i < 100) hansu++;
		else if (i < 1000) {
			a = i % 10;					//일의 자리
			b = (((i % 100) - a) / 10); //십의 자리
			c = i / 100;				//백의 자리

			if ((a + c) / (float)2 == (float)b) {
				hansu++;
			}
		}
	}
	return hansu;
}

int main() {
	int a = 0; 
	cin >> a;
	cout << count(a);
}
```
