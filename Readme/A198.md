## 문제 풀이

### 문제 링크
[14627. 파닭파닭 – 백준](https://www.acmicpc.net/problem/14627)

---

### 문제 설명
> 승균이는 파닭을 만들기 위해 파를 샀다. 파의 개수 `S`와 파닭 주문 수 `C`가 들어온다.
> 파닭 하나당 들어가는 파의 양을 일정하게 유지하고 싶어하며, 최대한 많은 양의 파를 넣고 싶어 한다.
> 각각의 파는 길이가 다르며, 파를 잘라서 사용할 수 있다. (단, 파를 이어 붙일 수는 없다.)
> 주문 받은 파닭 `C`개를 만들고 남은 파의 양(라면에 넣을 파)을 구하는 프로그램을 작성하시오.

---

### 접근 방식
1.  **이분 탐색 (Binary Search)**, 그 중에서도 **파라메트릭 서치 (Parametric Search)** 를 사용한다.
2.  우리가 찾아야 하는 것은 "파닭 하나에 넣을 수 있는 파의 최대 길이" (`best`)이다.
3.  파의 길이 `mid`를 정했을 때, 가지고 있는 `S`개의 파에서 `mid` 길이로 몇 개의 조각(`pieces`)을 만들 수 있는지 계산한다.
    -   `pieces >= C`: `C`개 이상 만들 수 있으므로, 길이를 더 길게 늘려본다. (`lo = mid + 1`, `best` 갱신)
    -   `pieces < C`: `C`개를 만들 수 없으므로, 길이를 줄여야 한다. (`hi = mid - 1`)
4.  **주의할 점**:
    -   입력되는 파의 길이와 총합이 `int` 범위를 넘을 수 있으므로 `long long`을 사용해야 한다.
    -   마지막에 남은 파의 양을 구할 때, `전체 파의 합(total) - (파닭 1개당 파의 길이(best) * 주문 수(C))`로 계산한다.
    -   만들어진 파 조각이 `C`개보다 많더라도, 우리는 딱 `C`개의 파닭만 팔고 나머지는 라면에 넣으므로 `best * C`만큼만 차감하는 것이 맞다.

---

### 풀이 코드

```cpp
#include <iostream>
#include <vector>

using namespace std;

int main() {

    int S, C;
    cin >> S >> C;

    vector<long long> len(S);
    long long hi = 0;
    long long total = 0;

    for (int i = 0; i < S; i++) {
        cin >> len[i];
        hi = max(hi, len[i]);
        total += len[i];
    }

    long long lo = 1;
    long long best = 0;

    while (lo <= hi) {
        long long mid = lo + (hi - lo) / 2; 
        long long pieces = 0;

        for (int i = 0; i < S; i++) {
            pieces += len[i] / mid;
            if (pieces >= C) break; 
        }

        if (pieces >= C) {          
            best = mid;
            lo = mid + 1;
        } else {                    
            hi = mid - 1;
        }
    }

    long long ramen = total - best * (long long)C;
    cout << ramen << "\n";
}
```
