## 문제 풀이

### 문제 링크
[2438. 별 찍기 - 1 – 백준](https://www.acmicpc.net/problem/2438)

---

### 문제 설명
> 첫째 줄에는 별 1개, 둘째 줄에는 별 2개, N번째 줄에는 별 N개를 찍는 문제입니다.
> 첫째 줄에 N(1 ≤ N ≤ 100)이 주어집니다.

**예제 패턴 (N=5인 경우)**
```
*
**
***
****
*****
```

---

### 접근 방식
1.  **반복문 사용**:
    -   바깥 루프 `i`는 행을 담당하며, 0부터 `N-1`까지 반복합니다.
    -   안쪽 루프 `j`는 별의 개수를 담당하며, 0부터 `i`까지 반복하여 `*`을 출력합니다.
    -   각 행이 끝날 때마다 줄바꿈(`\n`)을 수행합니다.
2.  **출력 형식**:
    -   `i=0`일 때 `j`는 0 (별 1개)
    -   `i=1`일 때 `j`는 0, 1 (별 2개)
    -   ...
    -   `i=N-1`일 때 `j`는 0 ~ N-1 (별 N개)

---

### 풀이 코드

```cpp
#include <iostream>
using namespace std;


int main() {
    int number;
    cin >> number;
    for (int i = 0; i < number; i++) {
        for (int j = 0; j <= i; j++) {
            cout << "*";
        }
        cout << "\n";
    }
}
```
