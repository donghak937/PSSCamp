## 문제 풀이

### 문제 링크
[나머지가 1이 되는 수 찾기 – 프로그래머스](https://school.programmers.co.kr/learn/courses/30/lessons/87390)
*(참고: 실제 프로그래머스 문제 번호와 파일명 A176은 사용자가 임의로 매핑한 것으로 보임. 코드 로직상 이 문제에 해당함)*

---

### 문제 설명
> 자연수 `n`이 매개변수로 주어집니다. 
> `n`을 `x`로 나눈 나머지가 1이 되도록 하는 가장 작은 자연수 `x`를 return 하도록 solution 함수를 완성해주세요. 
> 답이 항상 존재함은 증명될 수 있습니다.

**제한사항**
*   3 ≤ `n` ≤ 1,000,000

---

### 접근 방식
1.  **반복문 탐색**:
    -   우리가 찾아야 하는 것은 `n % x == 1`을 만족하는 가장 작은 `x`이다.
    -   `x`는 적어도 2 이상이어야 한다 (1로 나누면 나머지는 항상 0이므로).
    -   따라서 `x`를 1부터 시작하여 1씩 증가시키며 찾는다. (코드에서는 `i`를 사용)
2.  **조건 확인**:
    -   `n % i == 1`인 경우, `i`가 바로 그 답이 된다.
    -   가장 작은 수부터 차례대로 확인하므로, 조건을 만족하는 첫 번째 수가 무조건 최솟값이다.
    -   답을 찾으면 바로 `return`한다.

---

### 풀이 코드

```cpp
#include <string>
#include <vector>

using namespace std;

int solution(int n) {
    int i = 1;
    while(true){
        if ( n % i == 1) return i;
        i++;
    }
}
```
