## 문제 풀이

### 문제 링크
[없는 숫자 더하기 – 프로그래머스](https://school.programmers.co.kr/learn/courses/30/lessons/86051)
*(참고: 실제 프로그래머스 문제 번호와 파일명 A173은 사용자가 임의로 매핑한 것으로 보임. 코드 로직상 이 문제에 해당함)*

---

### 문제 설명
> 0부터 9까지의 숫자 중 일부가 들어있는 배열 `numbers`가 매개변수로 주어집니다.
> `numbers`에서 찾을 수 없는 0부터 9까지의 숫자를 모두 찾아 더한 수를 return 하도록 solution 함수를 완성해주세요.

**제한사항**
*   1 ≤ `numbers`의 길이 ≤ 9
*   0 ≤ `numbers`의 모든 수 ≤ 9
*   `numbers`의 모든 수는 서로 다릅니다.

---

### 접근 방식
1.  **전체 합 이용**:
    -   0부터 9까지의 모든 숫자의 합은 45입니다. (`0+1+2+...+9 = 45`)
    -   `numbers`에 없는 숫자들의 합은 `45 - (numbers의 모든 요소의 합)`과 같습니다.
2.  **사용자 코드 방식**:
    -   초기 `answer`를 45로 설정합니다.
    -   0부터 10까지 루프를 돌면서(`i`), 만약 `i`가 `numbers` 배열 안에 존재하면 `answer`에서 `i`를 뺍니다.
    -   (참고: 문제 조건상 숫자는 0~9이므로 반복문 범위가 `i < 11`인 것은 10까지 검사하지만 결과에 영향은 주지 않습니다.)

---

### 풀이 코드

```cpp
#include <string>
#include <vector>

using namespace std;

int solution(vector<int> numbers) {
    int answer = 45;
    for(int i = 0; i < 11; i++){
        for(int j =0; j < numbers.size(); j++){
            if(i == numbers[j]) answer-=i;
        }
    }
    return answer;
}
```
