## 문제 풀이

### 문제 링크
[폰켓몬 – 프로그래머스](https://school.programmers.co.kr/learn/courses/30/lessons/1845)

---

### 문제 설명
> N마리의 폰켓몬 중에서 N/2마리를 가져갈 수 있을 때, 가장 많은 종류의 폰켓몬을 선택하는 방법을 찾아 그때의 종류 번호 개수를 구하는 문제.

---

### 접근 방식
1. 폰켓몬의 종류를 파악하기 위해 배열을 정렬하고 `unique`, `erase`를 사용하여 중복을 제거
2. 선택할 수 있는 최대 마리 수(`choose = N/2`)와 중복을 제거한 폰켓몬 종류 수(`nums.size()`)를 비교
3. 종류 수가 선택 가능 수보다 많으면 `choose`를, 적으면 `nums.size()`를 반환 (즉, `min(종류 수, N/2)`)
---

### 풀이 코드

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main(){

    vector<int> nums = {3,1,2,3};

    int choose = nums.size() / 2;
    int answer;

    sort(nums.begin(), nums.end());                 
    nums.erase(unique(nums.begin(), nums.end()), nums.end());

    answer = nums.size() > choose ? choose : nums.size();

    return answer;
}
```
