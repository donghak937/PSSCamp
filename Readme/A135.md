##  문제 풀이

###  문제 링크  
[1302. 베스트셀러 – 백준](https://www.acmicpc.net/problem/1302)

---

###  문제 설명  
> 최빈값 찾기
---

###  접근 방식  
1. map을 이용해서 나온 수도 같이 카운트
2. max값과 같은 값의 key를 출력
---

### 풀이 코드

```cpp
#include <iostream>
#include <algorithm>
#include <map>

using namespace std;

int main() {
    int n;
    map<string, int> cnt;
    cin >> n;
    int mx = 0;
    for(int i = 0; i < n; i ++){
        string s;
        cin >> s;
        mx = max(mx, ++cnt[s]);
    }

    for (auto &p : cnt) {
        if (p.second == mx) {
            cout << p.first << "\n";
            break; 
        }
    }
}
```
