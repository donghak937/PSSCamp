## 문제 풀이

### 문제 링크
[같은 숫자는 싫어 – 프로그래머스](https://school.programmers.co.kr/learn/courses/30/lessons/12906)

---

### 문제 설명
> 주어진 배열에서 연속적으로 나타나는 숫자는 하나만 남기고 전부 제거한 뒤, 남은 수들을 반환하는 문제. (원소들의 순서 유지)

---

### 접근 방식
1. 배열의 모든 원소를 순회하며 결과 리스트에 담음
2. 현재 숫자가 바로 이전 숫자와 다를 경우에만 결과 리스트에 추가
3. 이전 숫자가 없는 첫 번째 원소는 항상 추가
---

### 풀이 코드

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main(){

    vector<int> arr = {1,1,3,3,0,1,1};
    vector<int> answer;

    for(int i = 0; i < arr.size(); i++){
        if(i == 0) answer.push_back(arr[i]);
        else if (arr[i-1] == arr[i]) continue;
        else answer.push_back(arr[i]);
    }

}
```
