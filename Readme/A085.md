##  문제 풀이

###  문제 링크  
[1431. 시리얼 번호 – 백준](https://www.acmicpc.net/problem/1431)

---

###  문제 설명  
> 문자열을 크기순, 안에 있는 숫자순, 알파벳 순으로 정렬한다.
---

###  접근 방식  
1. 사용자 정의 함수를 만든다.
2. isdigit를 써서 숫자만 더한다.
2. sort
---

### 풀이 코드

```cpp
#include <vector>
#include <algorithm>
#include <iostream>

using namespace std;

bool coms(string a, string b){
    if (a.size() == b.size()){
        int sumA = 0, sumB = 0;
        for (char c : a) if (isdigit(c)) sumA += c - '0';
        for (char c : b) if (isdigit(c)) sumB += c - '0';
        if (sumA == sumB){
            return a < b;
        }
        return sumA < sumB;
    }
    return a.size() < b.size();

}
  
int main() {
    
    int N;
    vector<string> a;
    cin >> N;
    for(int i = 0; i < N; i++){
        string tmp;
        cin >> tmp;
        a.push_back(tmp);
    }
    sort(a.begin(), a.end(), coms);
    for (const auto& s : a) {
        cout << s << '\n';
    }

}


```
