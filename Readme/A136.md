##  문제 풀이

###  문제 링크  
[1543. 문서 검색 – 백준](https://www.acmicpc.net/problem/1543)

---

###  문제 설명  
> 주어진 문서에서 특정 단어가 최대 몇 번 중복되지 않게 등장하는지 찾는 문제
---

###  접근 방식  
1. 문서 전체를 `getline`으로 입력받음
2. 찾고자 하는 단어를 `getline`으로 입력받음
3. `string::find`를 이용해 단어를 찾고, 찾은 경우 카운트를 증가시킨 뒤 해당 부분을 특수문자("##")로 치환하여 중복 검색을 방지함
---

### 풀이 코드

```cpp
#include <iostream>
#include <string>

using namespace std;

int main() {
  string s;
  getline(cin, s);

  string word;
  getline(cin, word);

  size_t pos = 0;
  int count = 0;
  while (pos != string::npos) {
    pos = s.find(word);
    if (pos != string::npos) {
      count++;
      s.replace(pos, word.size(), "##");
    }
  }
  cout << count;
}
```
