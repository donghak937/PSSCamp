## 문제 풀이

### 문제 링크
[10172. 개 – 백준](https://www.acmicpc.net/problem/10172)

---

### 문제 설명
> 아래 예제와 같이 개를 출력하시오.

**예제 출력**
```
|\_/|
|q p|   /}
( 0 )"""\
|"^"`    |
||_/=\\__|
```

---

### 접근 방식
1.  **이스케이프 시퀀스(Escape Sequence)** 처리:
    -   역슬래시(`\`)와 큰따옴표(`"`)를 출력하려면 앞에 역슬래시를 하나 더 붙여줘야 한다.
    -   `\` 출력 -> `\\`
    -   `"` 출력 -> `\"`
2.  **줄바꿈**:
    -   각 줄 끝에 `\n`을 넣어 줄바꿈을 한다.

---

### 풀이 코드

```cpp
#include <iostream>

using namespace std;

int main(){
    cout << "|\\_/|\n";
    cout << "|q p|   /}\n";
    cout << "( 0 )\"\"\"\\\n";
    cout << "|\"^\"\`    |\n";
    cout << "||_/=\\\\__|";
    
    return 0;
}
```
