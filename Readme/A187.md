## 문제 풀이

### 문제 링크
[2753. 윤년 – 백준](https://www.acmicpc.net/problem/2753)

---

### 문제 설명
> 연도가 주어졌을 때, 윤년이면 1, 아니면 0을 출력하는 프로그램을 작성하시오.
> 윤년은 연도가 4의 배수이면서, 100의 배수가 아닐 때 또는 400의 배수일 때이다.
> 예를 들어, 2012년은 4의 배수이면서 100의 배수가 아니라서 윤년이다. 1900년은 100의 배수이고 400의 배수는 아니기 때문에 윤년이 아니다. 하지만, 2000년은 400의 배수이기 때문에 윤년이다.

---

### 접근 방식
1.  **조건문 사용**:
    -   연도 `year`를 입력받는다.
    -   문제에서 제시한 윤년 조건을 그대로 코드로 옮긴다.
    -   `(year % 4 == 0 && year % 100 != 0) || year % 400 == 0`
        -   4의 배수(`% 4 == 0`)이고(`&&`), 100의 배수가 아닐 때(`% 100 != 0`)
        -   또는(`||`)
        -   400의 배수일 때(`% 400 == 0`)
2.  **출력**:
    -   조건이 참이면 `1`, 거짓이면 `0`을 출력한다.

---

### 풀이 코드

```cpp
#include <iostream>

using namespace std;

int main (){
    int year;
    cin >> year;
    
    if ((year % 4 == 0 && year % 100 != 0) || year % 400 == 0 ){
        cout << "1";
    }
    else{
        cout << "0";
    }
    
}
```
